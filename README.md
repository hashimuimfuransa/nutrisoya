# Nutrisoya - Healthy Soya Products

Healthy Living Starts with Soya 🌱

This is the official website for Nutrisoya, featuring affordable, nutritious, and delicious soya products for everyone.

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Open your browser and navigate to `http://localhost:3000`

## Features

- 🌱 Modern, responsive design
- 🛒 Product catalog with WhatsApp ordering
- 🌍 Bilingual support (English & Kinyarwanda)
- 📱 Mobile-first approach
- ⚡ Fast and optimized performance

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Motion (Animations)
- Lucide React (Icons)
